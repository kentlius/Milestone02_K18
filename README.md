# Manual (Development)
Install npm atau yarn setelah clone pertama kali
## NPM
Setelah clone,
```
npm i
```
Tengok aplikasi
```
npm start
```
## yarn
setelah clone,
```
yarn add .
```
Tengok aplikasi
```
yarn serve
```
